# 과천중앙고등학교 시간표 웹사이트

과천중앙고등학교 시간표를 누구나 쉽게 조회할 수 있는 GitHub Pages 웹사이트입니다.

## 기능
- 교사 시간표 검색
- 학년별 시간표 검색
- 최근 조회 기능
- 반응형 디자인

## 배포 주소
https://aaronlee09-max.github.io/gwacheon-timetable/
https://aaronshlee.run.place/

## 사용법
1. GitHub Pages 자동 배포 완료
2. 사이트 접속
3. 검색을 이용해 원하는 교사/학년 선택

## 개발자
```bash
git add .
git commit -m "Update timetable"
git push origin main
```

이 프로젝트는 Python generate.py로 자동 생성됩니다.
